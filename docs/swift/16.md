# Enum 枚举

## 例子

使用 `enum` 和 `case` 声明枚举

```swift
// 首字母大写
enum Weekday {
  case Mon
  case Tue
  case Wed
  case Thu
  case Fri
  case Sat
  case Sun
}

var a = Weekday.Mon

print(a) // Mon
print(type(of: a)) // Weekday
```

也可以使用单行输入的方式

```swift
enum Position {
  case top, right, bottom, left
}

var a = Position.right

print(a) // right
print(type(of: a)) // Position
```

## 修改值

如果想要修改一个枚举，可以使用只要 `.` 即可修改

```swift
enum Position {
  case top, right, bottom, left
}

var a = Position.right

a = .top

print(a) // top
```

## 匹配

可以使用 `.` 直接快速匹配

```swift
enum Position {
  case top, right, bottom, left
}

var a = Position.right

switch a {
  case .bottom:
  print("下")

  case .top:
  print("上")

  case .right:
  print("右")

  case .left:
  print("左")
}
```

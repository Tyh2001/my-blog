## 运算符

## 基本

```swift
var a = 1

print(-a) // 一元运算符

print(1 + 1) // 二元运算符

print(1 > 2 ? 3 : 0) // 三元运算符
```

## 赋值运算符

```swift
var a = 1


var (b,c) = (10,20)
```

## 空合运算符

默认值是空的，设置类型 值为 nil

```swift
var h:String? = nil

print(h ?? "")
```

## 区间运算符

```swift
for index in 1...10 {
    print(index,terminator: " ")
}
```

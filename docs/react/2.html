子组件给父组件传值 ​

父组件给子组件传递一个方法，子组件调用父组件方法，将需要修改的值以参数的形式传递给父组件

tsx
import { Component } from 'react'

const Header = (prop) => {
  const onChange = (e) => {
    prop.change(e.target.value)
  }

  return (
    <>
      <h1>{prop.value}</h1>
      <h1>{prop.text}</h1>
      <input type="text" onChange={onChange} />
    </>
  )
}

export class App extends Component {
  state = {
    value: ''
  }

  changeText = (value) => {
    this.setState({ value })
  }

  render() {
    return (
      <>
        <h1>{this.state.value}</h1>
        <Header change={this.changeText} />
      </>
    )
  }
}

生命周期 ​

只有 extends React.Component 的组件才会有生命周期。

  • 组件初始化阶段 constructor
  • 组件加载阶段 componentWillMount
  • 组件更新阶段 render
  • 组建加载完成阶段 componentDidMount
tsx
import React from 'react'

class Button extends React.Component {
  constructor(props: { text: string }) {
    super(props)
    this.state = {
      ...props
    }

    console.log('1. 组件初始化')
  }

  componentWillMount() {
    console.log('2. 组件加载之前')
  }

  render() {
    console.log('3. 组件加载或者更新')
    return <button>{this.state.text}</button>
  }

  componentDidMount() {
    console.log('4. 组件加载之后，可以获取到 DOM 节点')
  }
}

export class App extends React.Component {
  render() {
    return (
      <div>
        <Button text="按钮" />
      </div>
    )
  }
}

高阶组件 HOC ​

和高阶函数的感念一样，传入一个函数组件，返回一个新的组件

tsx
import { Component, ReactNode } from 'react'

const PropsButton = (WrapCompoent) => {
  return class extends Component {
    state = {
      text: ''
    }

    componentDidMount(): void {
      this.setState({ text: '按钮' })
    }

    render(): ReactNode {
      return <WrapCompoent buttonText={this.state.text}></WrapCompoent>
    }
  }
}

class _Button extends Component {
  render() {
    return <button>{this.props.buttonText}</button>
  }
}

const Button = PropsButton(_Button)

export class App extends Component {
  render() {
    return (
      <div>
        <Button />
      </div>
    )
  }
}

调用接口拿数据展示 ​

tsx
import { Api } from '../../api'
import { useState, useEffect } from 'react'

const API = new Api()

export const UsersDoctor = () => {
  const [doctorList, setDoctorList] = useState<any>([])

  const doctorlist = async () => {
    const res = await API.doctorlist({
      pageNumber: 1,
      pageSize: 20
    })

    if (res.code === '1') {
      const resultList = res.data.results.map((item: any, index: number) => {
        return <div key={index}>{item.doctorId}</div>
      })
      setDoctorList(resultList)
    }
  }

  useEffect(() => {
    doctorlist()
  }, [])

  return (
    <div>
      <h1>列表</h1>
      <div>{doctorList}</div>
    </div>
  )
}

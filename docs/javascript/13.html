访问器 ​

在正常对象中，对象中的属性我们是可以随意设置和更改的，但是有些时候并不希望某些值被设置了不可控的值，比如：

js
const user = {

  age: 12

所以就需要加以限制，需要在对象中新建两个获取函数，分别使用 set 和 get 声明，那么每次获取和修改都会经过这里，来进行判断

js
const user = {
  data: {
    name: '张同学',
    age: 12
  },
  set age(val) {
    if (typeof val !== 'number' || val < 1 || val > 100) {
      throw new Error('年龄格式错误')
    }
    this.data.age = val
  },
  get age() {
    return this.data.age
  }
}
